# Clasificador Gaussian Naive Bayes
Clasificador Gaussiano Bayesiano programado en python.
El clasificador emplea el metodo de validación K-fold con k = 10 y si se desea emplear leave-one-out recordar que k=n, donde n es la cantidad de datos del dataset.
Las metricas de evaluación usadas son: Accuracy, Precision, Recall, F1-Score, Balanced Accuracy y Specificity
Los conjuntos de datos de entrada pueden ser .csv y .arff
